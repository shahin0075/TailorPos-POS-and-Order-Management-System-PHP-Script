Reports
Eg: Daily Sales Report
There are many reports available in Tailor POS to enable you to monitor your business and crucial information regarding the finances which helps you take better decisions.
The reports available to the branches and admin differ.Branches can only see their own data, while admin can view data of branches. The branch filter option is also visible for the admin.
Branch Reports :
  1. Daily Sales Report
  2. Sales Report
  3. Payment Report
  4. Expense Report
  5. Sales Return Report
  6. Day-Wise Report
  7. Branch Stock Report
  8. Income Report
Admin Reports :
  1. Daily Sales Report
  2. Sales Report
  3. Payment Report
  4. Expense Report
  5. Sales Return Report
  6. Purchase Report
  7. Purchase Payments Report
  8. Stock Report
  9. Day-Wise Report
  10. Income Report
